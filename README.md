## About me

I'm a curious builder based in Kenya, studying **Information Technology at [Zetech University](https://www.zetech.ac.ke/)** and growing across software engineering, cybersecurity, web development, Android, data, and machine learning.

I like projects that teach me something new, break nicely when I test them, and eventually become useful enough for real people. Still a wannabe haxor, but a serious full-time learner.

- Completed the **[ALX Africa Software Engineering Bootcamp](https://www.alxafrica.com/)** from Jan 2022 to Jan 2023
- Currently sharpening my skills in **secure software development**, **backend systems**, and **practical data workflows**
- Open to collaborating on tools, experiments, CTF-style projects, dashboards, and wonderfully odd ideas
- Powered by code, curiosity, and a very reasonable amount of chai
